Contexto escolar

O Desafio do Ritmo de Aprendizagem

Fadiga Cognitiva

Exposições teóricas longas e lineares esgotam a capacidade de retenção natural do estudante ao longo do dia letivo.

Limitações de Ensino

Impossibilidade física de monitorizar e adaptar o ensino individualmente a ritmos diferentes de múltiplos alunos em simultâneo.

Lacunas Cumulativas

Pequenas dúvidas não esclarecidas a tempo geram atrasos na matéria que comprometem o aproveitamento final.

Resiliência e Auto-Recuperação

Algoritmo de retry com recuo exponencial. Se a API de Inteligência Artificial falhar temporariamente, o sistema recupera-se autonomamente sem crashar.

Injeção de Contexto Dinâmica

O serviço lê o perfil do utilizador no Supabase e reconstrói o prompt em tempo real, garantindo interações 100% personalizadas ao neurotipo do estudante.

Fluxo de Engenharia

Ciclo de Comunicação Híbrido

1. Input

O utilizador submete uma dúvida através da interface reativa em React e Tailwind CSS.

2. Orquestração

O Electron gere processos nativos, enquanto o navegador efetua requisições HTTPS diretas via API.

3. IA Hub

O OpenRouter encaminha o pedido para o modelo LLM fixo configurado no código (DeepSeek).

4. Nuvem

Os dados de evolução são sincronizados e gravados instantaneamente no Supabase.

Frontend

Desenvolvimento de Interface Reativa

React JS

Biblioteca baseada em componentes que garante o reaproveitamento total do código visual no navegador e no desktop.

Analogia: Blocos de construção estáveis que se encaixam perfeitamente em qualquer ecrã.

Tailwind CSS

Framework de classes utilitárias que permite criar um design performativo e visualmente consistente.

Analogia: Um alfaiate com moldes pré-medidos, garantindo rapidez e precisão estética.

Ambiente

Isolamento vs Acessibilidade

Electron JS

Encapsula a aplicação num software nativo, criando um ecossistema de estudo isolado e livre de distrações.

Analogia: Uma moldura protetora que transforma o site num programa real local.

Navegador Web

Execução direta no browser, permitindo flexibilidade total para o aluno estudar sem instalações prévias.

Analogia: Aceder a um serviço via bilheteira pública aberta: instantâneo e global.

Dados

Persistência e Infraestrutura Cloud

Sincronização Cloud

Assegura que todo o percurso — notas e progresso — não se perca quando a aplicação é encerrada.

Analogia: Um caderno que escreve sozinho numa biblioteca central em nuvem, mantendo tudo seguro.

Supabase

Base de dados relacional robusta que gere o armazenamento estruturado e o sistema de autenticação.

Analogia: Um banco altamente organizado com cofres individuais: dados blindados e indexados.

Segurança

Encriptação e Blindagem de Dados

Autenticação JWT

Tokens cifrados que validam a identidade do aluno em cada pedido sem expor a palavra-passe na rede.

Hashing Robusto

As credenciais nunca são guardadas em texto limpo, sendo processadas por funções criptográficas seguras.

Segurança RLS

Row Level Security: Isolamento absoluto onde um utilizador só consegue interagir com as suas próprias linhas.

Engenharia

Métricas de Latência e Eficiência

Latência (<5s)

Tempo total entre o envio do pedido e a resposta da IA. Manter este valor baixo respeita a biologia da atenção.

Refatoração Local

Limpeza de algoritmos JavaScript para garantir excelente fluidez mesmo em computadores escolares antigos.

Testes

Controlo de Qualidade Multiplataforma

Bug Detetado

Inconsistência de Emojis em ambiente Linux. Os ícones nativos do Windows ficavam corrompidos.

Resolução

Desenvolvimento de um "Modo Humor" nas definições para desativar emojis e manter a funcionalidade.

Rigor Técnico

Documentação que prova testes estritos em ambientes Windows e Linux.
