Contexto

O Desafio do Ritmo Escolar

Fadiga

Aulas longas, lineares e contínuas saturam a retenção natural do estudante ao longo do dia.

Limites

Impossibilidade física de um único docente validar e acompanhar individualmente o ritmo de 20 ou mais alunos em simultâneo.

Lacunas

Dúvidas pontuais não sanadas a tempo geram atrasos complexos e cumulativos ao longo do ano letivo.

Proposta

TutorAI: A Solução Inteligente

Adaptativo

Ajuste automático da complexidade dos exercícios com base no histórico direto e nas reais dificuldades detetadas.

Didático

Uso de analogias práticas do quotidiano e dinâmicas de gaming para explicar conceitos puramente abstratos.

Multiplataforma

Disponível tanto como aplicação nativa de Desktop para máximo isolamento, quanto via Navegador Web para acesso rápido.

Métricas e Evolução

O Reflexo do Aluno: Domínio e Progresso

Dashboard de Precisão

Exibição analítica imediata da percentagem de acertos e erros do utilizador distribuída pelas quatro disciplinas base: Português, Matemática, Inglês e História.

O Baú do Conhecimento

Espaço de consolidação. Os testes realizados com sucesso no Chat Inteligente são arquivados e listados como "domínio conquistado" na interface do aluno.

Fluxo Técnico

Ciclo de Comunicação Híbrido

1. Input Local

O utilizador interage de forma fluida com a interface construída com blocos em React + Tailwind CSS.

2. Execução

Ambiente Desktop: O Electron processa a chamada local nativa. / Ambiente Web: O browser faz as requisições HTTPS normais.

3. IA Hub

O OpenRouter intermedeia e escolhe inteligentemente o melhor modelo de IA remoto para gerar o feedback ideal.

4. Cloud & Dados

Tudo é sincronizado e guardado de forma persistente e segura na infraestrutura de base de dados do Supabase.

Frontend

Interface e Design

React JS

Biblioteca para construir interfaces modulares. É a base única do ecossistema, permitindo reaproveitar 100% do design tanto no ambiente web quanto no desktop.

Analogia: Peças de Lego independentes que se encaixam e funcionam em qualquer tabuleiro (browser ou app).

Tailwind CSS

Framework focado em design utilitário. Elimina redundância e arquivos pesados, garantindo uma aplicação leve e padronizada visualmente.

Analogia: Um alfaiate com moldes prontos, garantindo rapidez e precisão extrema no design.

Ambiente

Nativo Desktop vs Execução Web

Electron JS (Desktop)

Runtime que encapsula a interface num executável nativo (.exe). Cria um ecossistema focado e isolado para evitar que distrações da navegação comum prejudiquem o foco do aluno.

Analogia: Uma moldura que transforma o site num programa real instalado localmente no PC.

Navegador (Web)

Execução direta no browser. O código React corre na engine local (V8/SpiderMonkey) do próprio navegador do utilizador, dispensando instalações e facilitando o acesso rápido.

Analogia: Aceder a um serviço público através de uma bilheteira geral sem precisar de uma chave privada.

IA Hub

Protocolos de Dados e IA

O que são APIs?

Pontes de comunicação reguladas que permitem que o TutorAI envie pedidos locais e receba respostas rápidas de servidores de inteligência artificial remotos.

Analogia: O empregado de mesa que recolhe o pedido e traz a refeição pronta da cozinha.

OpenRouter

Gateway inteligente de IA. Se um modelo falhar ou ficar lento, o sistema muda o tráfego de forma automatizada e impercetível (Failover estrutural).

Analogia: Uma central telefónica inteligente que te conecta instantaneamente ao melhor perito disponível.

Backend

Persistência e Segurança

Persistência na Cloud

Gravação durável de dados em servidores dedicados. Garante que os níveis de domínio e notas do aluno sobrevivam estáveis ao fecho do programa.

Analogia: Um cofre virtual infinito e seguro, sempre sincronizado via Web.

Supabase

Infraestrutura estável assente em PostgreSQL. Responsável pela gestão relacional do histórico e armazenamento seguro de tokens de sessão do utilizador.

Analogia: Um sistema bancário com cofres privados invioláveis para guardar as contas dos utilizadores.

Segurança

Cifragem e Proteção de Dados

JWT

Tokens cifrados que validam e seguram a identidade do aluno a cada ciclo de uso, eliminando a exposição direta de dados confidenciais.

Hashing Robust

Passwords guardadas com chaves criptográficas de sentido único, tornando as credenciais completamente ilegíveis no banco de dados.

RLS (Row Level Security)

Isolamento estrito ao nível da tabela do Supabase. Cada estudante só consegue ler ou editar exclusivamente os seus próprios registos.

Performance

Latência e Otimização

Latência (<5s)

Métrica de atraso físico na transmissão de dados da IA. Manter este valor baixo previne a quebra de atenção e o cansaço do estudante no processo ativo de estudo.

Analogia: O tempo exato que decorre entre o clique no interruptor e a lâmpada acender.

Otimização

Refatoração do JavaScript e queries limpas para garantir o menor consumo possível de CPU e memória RAM local, permitindo rodar em computadores escolares antigos.

Analogia: Aliviar o peso morto de um carro para ele andar mais rápido gastando menos.
